Teoria da Computação

Regras para Usuário:

O usuário deve ser composto por no máximo 10 caracteres. Não pode conter números nem letras maiúsculas.

Regras para Senha:

A senha deve ter no mínimo 8 caracteres, incluindo pelo menos um caractere especial (@#$%¨&) e um número (0123456789).

Autômato Finito Determinístico (AFD) para Usuário:

  • Conjunto de Estados: Q = { Q0, Q1, Q2, Q3, Q4, Q5, Q6, Q7, Q8, Q9 }
  • Alfabeto: Σ = { a, b, c, ..., z }
  • Função de Transição:
    • Δ(q0, u) = q1
    • Δ(q1, s) = q2
    • Δ(q2, u) = q3
    • Δ(q3, a) = q4
    • Δ(q4, r) = q5
    • Δ(q5, i) = q6
    • Δ(q6, o) = q7
    • Δ(q7, s) = q8
  • Estado Inicial: Q0
  • Estado Final: Q8

Autômato Finito Determinístico (AFD) para Senha:

  • Conjunto de Estados: Q = { Q0, Q1, Q2, Q3, Q4, Q5, Q6, Q7, Q8, Q9 }
  • Alfabeto: Σ = { a, b, c, ..., z, 0, 1, 2, ..., 9, !, @, #, $, & }
  • Função de Transição:
    • Δ(q0, u) = q1
    • Δ(q1, s) = q2
    • Δ(q2, u) = q3
    • Δ(q3, @) = q4
    • Δ(q4, r) = q5
    • Δ(q5, i) = q6
    • Δ(q6, o) = q7
    • Δ(q7, s) = q8
    • Δ(q8, 1) = q9
    • Δ(q9, 2) = q10
    • Δ(q10, 3) = q11
  • Estado Inicial: Q0
  • Estado Final: Q11
